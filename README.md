# 🥩 Frigorífico Bah | E-commerce Gastronômico

> "A excelência em cada corte, a tecnologia em cada clique."

O **Frigorífico Bah** é uma plataforma de e-commerce de alta performance, projetada para proporcionar uma experiência imersiva e intuitiva na jornada de compra de cortes bovinos premium. O sistema integra visualização interativa com um fluxo de checkout otimizado, conectando a tradição gaúcha à modernidade digital.

## 🚀 Arquitetura e Diferenciais

* **Mapa Interativo de Cortes:** Implementação de navegação visual via SVG, garantindo uma UX intuitiva para identificação de produtos.
* **Checkout Conversacional:** Fluxo de carrinho integrado que centraliza o pedido no WhatsApp, reduzindo a fricção e acelerando a conversão do usuário.
* **Performance First:** Aplicação de estratégias de *code-splitting* e otimização de ativos, resultando em carregamento rápido mesmo em dispositivos com rede instável.
* **Design System:** Interface construída com foco em acessibilidade e contraste, respeitando a identidade sóbria e premium da marca.

💻 Instalação e Execução
Pré-requisitos: Node.js v18+ e npm/yarn/pnpm.

Bash
# Clone o repositório
git clone [https://github.com/fonseca-felix/frigorifico-bah.git](https://github.com/fonseca-felix/frigorifico-bah.git)

# Acesse o diretório
cd frigorifico-bah

# Instale as dependências
npm install

# Inicie o ambiente de desenvolvimento
npm run dev
Abra http://localhost:3000 no seu navegador para visualizar.
