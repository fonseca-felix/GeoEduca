Stack Tecnológico

Tecnologias usadas no Frontend e no Backend da plataforma

Frontend

Interface construída com tecnologias web puras — sem frameworks pesados. Rápido, leve e fácil de manter.

HTML5 CSS3 Vanilla JavaScript ES6+ Font Awesome Google Fonts

Backend

Servidor RESTful em Node.js com Express. Gerencia autenticação JWT, rotas da API e integração com banco de dados.

Node.js Express.js JWT REST API Middleware

Banco de Dados

Firebase Firestore como banco NoSQL em tempo real. Armazena usuários, turmas, atividades e resultados com segurança.

Firebase Firestore NoSQL Tempo Real Google Cloud

Inteligência Artificial

Integração com o Google Gemini AI para geração de aulas, flashcards, quizzes e curadoria de vídeos sobre qualquer tema de Geografia.

Google Gemini Gemini 2.0 Flash Geração de Texto API REST

Autenticação & Segurança

Sistema de login seguro com tokens JWT. Separação de perfis entre Professor e Aluno, com rotas protegidas no servidor.

JWT bcrypt CORS Middleware Auth

Design System

Sistema de design próprio com variáveis CSS (tokens), modo escuro automático, componentes reutilizáveis e design responsivo.

CSS Tokens Dark Mode Responsivo Acessível

Comunicação via REST Frontend e Backend conversam via chamadas HTTP com JSON, usando o padrão REST.
JWT em cada requisição Toda chamada ao servidor carrega um Token de autenticação para verificar a identidade do usuário.
Dados em Tempo Real O Firestore atualiza listas de alunos, resultados e atividades sem precisar recarregar a página.
IA com Fallback O sistema tenta múltiplos modelos Gemini automaticamente se um deles estiver sobrecarregado.

O que professores e alunos podem fazer no GeoEduca

Área do Professor

Gerenciar Turmas e AlunosCriação de salas, matrícula e controle de alunos por turma.
Criar Atividades e ProvasMontagem de avaliações com questões dissertativas e objetivas.
Ver ResultadosAcompanhamento de desempenho individual e por turma em quizzes e jogos.
Gerenciar JogosAtivar e monitorar o BrazilGuessr e o Missão Brasil para as turmas.

Área do Aluno

Rota 27 — Estudos com IAGeração de aulas completas, flashcards, quizzes e sugestão de vídeos sobre qualquer tema geográfico.
BrazilGuessrJogo de adivinha a localização geográfica no mapa do Brasil a partir de pistas visuais.
Missão BrasilQuiz interativo sobre bandeiras, capitais e biomas dos estados brasileiros.
Responder AtividadesAcesso e entrega de atividades e provas criadas pelo professor.

Desenvolvido como Trabalho de Conclusão de Curso

GeoEduca foi criado com o objetivo de tornar o ensino de Geografia mais dinâmico, interativo e eficiente, combinando tecnologia moderna com metodologias pedagógicas atuais.
